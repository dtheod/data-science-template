# data-science-template
Data Science Template for DS projects

## Features

- **FastAPI**: Ready-to-use API endpoint for serving predictions.
- **Docker Support**: Includes `Dockerfile` and `docker-compose.yaml` for containerization.
- **Dependency Management**: Supports `uv` (ultra-fast) and `pip`.
- **Configuration**: Uses [Hydra](https://hydra.cc/) for flexible configuration management.
- **Validation**: Uses [Pydantic](https://docs.pydantic.dev/) for data validation.
- **Documentation**: Auto-generated API documentation with [pdoc](https://pdoc.dev/).
- **Code Quality**: Pre-configured [ruff](https://docs.astral.sh/ruff/), [black](https://github.com/psf/black), and [mypy](https://mypy-lang.org/) hooks.
